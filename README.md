## Exploratory Data Analysis

The exploratory data analysis focuses on understanding the cleaned dataset before performing detailed water quality analysis.

Current EDA includes:

- Dataset overview
- River distribution
- Water quality parameter distribution
- Parameter category analysis
- Sampling month distribution
- Interactive geographic visualization using Folium
